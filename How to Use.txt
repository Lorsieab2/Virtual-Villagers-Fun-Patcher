VIRTUAL VILLAGERS FUN PATCHER

1. Double-click "Launch Virtual Villagers Fun Patcher.bat".
2. Choose one of the two patch styles:
   - Collection Progression Max Pop:
     Collections still raise the cap and are needed to reach the absolute
     maximum. The Secret City also retains its level-3 magic bonus.
   - Immediate Fixed Max Pop:
     The absolute maximum is available immediately. Collections no longer
     affect it, and The Secret City's magic tech no longer affects it.
3. Optional for The Lost Children: check "Easier Healing Mastery".
4. Open "One Game" or "All 5 Games".

PATCH ALL FIVE AT ONCE

1. Use the five separate fields to choose one folder per game. Each folder must
   contain that game's ORIGINAL EXE.
2. Or use "Find All 5 in Parent Folder..." when the EXEs are in the selected
   folder or one folder below it.
3. Click "Validate All 5", "Dry Run All 5", or "Patch All 5".

PATCH ONE GAME

1. Browse to one ORIGINAL game EXE.
2. Click Validate, Dry Run, or Create Modified EXE.

OUTPUTS

Collection Progression creates:
(name of game) - Modified Max Pop.exe

Immediate Fixed creates:
(name of game) - Fixed Max Pop.exe

When Easier Healing Mastery is selected for VV2, its output name adds:
+ Easier Healing.exe

Each output and verification log is placed beside its original game EXE. Both
styles can coexist. Original EXEs are never changed, renamed, replaced, or
deleted. The patcher remembers the selected style, single-game EXE path, and all
five folder paths.

SAFE MULTIPLE BIRTHS

All five games normally check the cap before deciding singleton/twins/triplets.
Both styles prevent overflow at the real slot ceiling:
- 3+ spaces: multiples work normally.
- 2 spaces: a rolled triplet becomes twins.
- 1 space: a rolled twin or triplet becomes a singleton.
- 0 spaces: the normal cap check blocks the birth.

Only the five exact researched stock builds are supported. Unknown, modified,
corrupt, duplicate, or incorrectly assigned EXEs are refused.

VV2 EASIER HEALING MASTERY

When a healer or a villager who prefers Healing has no sick villager to treat,
the patch sends that villager into the game's existing plant-study work state.
That persistent state is used during normal play and time catch-up. Normal
healing, sickness, food, skill gains, and plant availability are unchanged.
